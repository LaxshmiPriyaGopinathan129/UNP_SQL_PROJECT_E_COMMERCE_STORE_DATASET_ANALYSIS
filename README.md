# UNP_SQL_PROJECT_E_COMMERCE_STORE_DATASET_ANALYSIS

DESCRIPTION: Designed and implemented a complete E-Commerce database system using SQL at UNP Education. The project includes structured tables for customers, products, orders, payments, shipping, and delivery with proper relationships. Performed CRUD operations, advanced joins (INNER, LEFT, RIGHT, FULL), aggregations (SUM, AVG, COUNT), and business analysis queries. Generated insights like total revenue, average order value, preferred payment modes, and top-performing products. Demonstrates practical database management and real-world business problem-solving skills.
